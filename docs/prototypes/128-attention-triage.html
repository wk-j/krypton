A · Backpressure gauge — the only ambient signalships v1

A single static count of open judgement items, living in the workspace status bar. It shows queue depth so you can scale the fleet to your own review rate — it never blinks, pulses, or turns an alert color. Hidden when the count is zero. This is not a notification.

3 lanes · 1 busy ◆3judgement

When zero: ◆ the segment simply disappears — no "all clear" badge, nothing to glance at.

B · Triage overlay — summon with leader jships v1

A summon-on-demand overlay (like the command palette) over the dimmed harness — never a persistent dashboard. Cards are ranked by reversibility / blast-radius (irreversible first), tie-broken by age (oldest first). Lane peek heat is not consulted — it measures busy-ness, not judgement weight. Every card always renders traded-off and unsure blocks (never collapsed) so you see what the agent gave up.

C · Redirect — r on a cardships v1

Redirect opens a one-line correction. It is delivered on the lane's next idle via the existing InterLaneCoordinator.enqueueSystemPrompt path — it does not interrupt the running turn. The lane figures out its own rework; late-arrival rework is the accepted price of non-blocking.
